HR Guide

Smooth, clear instructions for clan HRs

Verification Tickets

  • Accept players into the group and update them with Rover.
  • If not verified with Rover, instruct them to do so.
  • Assign timezone roles:
    • EST → Americas
    • GMT → Europe, Africa, Middle East
    • Asia → Asia & Oceania
  • Save the ticket transcript before closing and deleting it.
  • Note: We do NOT accept CW cases.

Raid Request Tickets

  • Give the requester the raid request role.
  • Do not touch the ticket after that.

Blacklists

Clan Blacklist

Format for #group-blacklists:

- Clan name:
- Time:
- Reason:

Player Blacklist

Pban them in-game, then use this format in #player-bans:

- Player name:
- Time:
- Reason:

Leader will review times and reasons. Use common sense.

Events

Shout events using the following format:

@everyone

# Defending/Raiding (clan name)

Gamemode:
Timezone:
Numbers: (e.g. 6+ 7+ 8+ 9+ etc.)

Link:
  • Before every raid/defense ends, take 1 screenshot of the player list (win or loss).
  • If victorious, also take a screenshot of the victory screen.
  • Post the victory screen screenshot in #victories and state the topfrag and secondfrag as MVPs.
  • Use the player list screenshot to give XP via /add xp in #rankings.

XP Distribution

  • 15XP → Topfrag & secondfrag in raids/defenses
  • 10XP → Each participating player
  • 5XP → Spectators (subs)
  • 5XP → Each player in DTS’s
  • Note: HR’s and Elites cannot receive XP.

Elite Logs

  • Log elite progress with format: topfrag/secondfrag over total events fully played.
  • Promote elites after hitting 5 topfrag/secondfrags.
  • If more than 2 elites attend, only log the topfrag & secondfrag — do not count it as a fail for others.

Promotion Note

If you are an active and knowledgeable HR, you will be promoted.
